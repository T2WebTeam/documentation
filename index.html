Must Take Note:

  1. Do not upload differect project's related materials to current project. Eg:
    • Images/Gif/Video
    • Language file
    • Project name (eg. copyright)
  2. Double check when importing LANGUAGE file in super admin.
  3. Do not create account in LIVE site. Use DIRECTOR account if test is needed.
  4. Do not tune code directly in MASTER/LIVE branch

Before start the task, you need to:

  • Understand the task requirement
  • Think of the logic and flow
  • Make sure API is ready with required data

When doing the task, you need to:

  • Make sure the logic is correct
  • Write clean and efficient code
  • Write dynamic code if possible. Know how to determine which code can be dynamic/hardcode
  • Indent properly
  • Comment temporary unused code (which is useful)
  • Remove unwanted code (check whether can remove or not first)
  • Check console make sure no error
  • Remove console.log data
  • Cannot display client id / id

After done the task and before update task to exec, you need to:

  1. Layout
    • No alignment issue
    • No responsive view issue
  2. Language
    • Language is correct
    • Plugin all language
    • Display data-lang
  3. Buttons, hyperlink
    • Make sure all buttons / hyperlink are working
  4. Data
    • All data display correctly
    • If data is null, add checking
    • Display "no result found" or "-" if no data
    • Numeric value add thousand separator
    • Calculation is correct (add NaN & negative value checking)
  5. Form Plugin
    • For numeric input, disallowed alphabet input
    • Display all error message
    • Can submit the form successfully
    • Clear the input, refresh the data / page when success submit the form
  6. Images, attachment, video
    • Make sure DO is working
    • Can upload, download and display
  7. Minimize possible bug, check with different test case (ask what if logic)
    • eg. What if successfully register more than 1 account with same username?
    • eg. What if my account can change other account password?
    • eg. If hide this page in sidebar, can user still view this page when type in url?
    • eg. If user account balance is 0, can user still proceed to do withdrawal?
    • eg. If tune this line of code, will it affect other function?
  8. Do final checking in TEST SITE
